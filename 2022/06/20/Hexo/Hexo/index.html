Hexo配置和使用

2022年06月20日
Hexo博客
留言
  1. 安装脚手架
  • 脚手架
npm install hexo-cli -g
  1. 生成文件
  • init
hexo init blog
  • 安装依赖
cd blog
npm install
  1. 生成网站
hexo g

过后会生成 public 目录

  • 启动网站
hexo server
  1. 命令
hexo new "text title"  // 生成md文章
hexo new draft "" // 草稿文章
hexo new page ""  //纯页面-  存入scaffolds中
  1. 配置
  • Front Matter
---
title: ""
category: //分类
	- 1
	- 2
conmment: on // 是否启用评论
---
  1. 上传到 github

  • 搭建仓库 username.github.io 命名
    username 必须跟 github 用户名一致

  • 安装依赖

yarn add hexo-deployer-git --save
  • _config.yml 配置
# Deployment
## Docs: https://hexo.io/docs/one-command-deployment
deploy:
  type: git
  repo: git@github.com:isAllyn/xiaolblog.github.io.git
  branch: master
  • 代码高亮
# 代码高亮配置文件
highlight:
  enable: true
  line_number: true
  auto_detect: false
  tab_replace: ''
  wrap: true
  hljs: false
prismjs:
  enable: false
  preprocess: true
  line_number: true
  tab_replace: ''
  1. 置顶
    置顶的文章的 Front-matter 中加上 top: true 即可
---
title: BlueLake博客主题的详细配置
tags: [hexo,BlueLake]
categories: Hexo博客
top: true
---
